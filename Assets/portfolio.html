Meally Food Delivery Tracker

A full-stack MCV application designed to be used by delivery drivers or restaurants to track outgoing food orders. Users are able to monitor the orders from creation to delivery and easily see the orders's stage, whether it has been picked up by a driver and if it has been delivered. This application Node.js, Sequelize ORM, and RESTful APIs as part of the back-end and renders data to the front-end through Handlebars templating engine. For user authentication, this application uses Passport.js OAUTH.
